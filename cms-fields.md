# CMS fields proposal – CTA Section Two Blocks

This document describes the proposed CMS fields for the CTA section.  
The goal is to allow marketers to easily edit the content without breaking the design.

---

## Label

- **Field label:** Label
- **Required:** Optional
- **Default value:** Label
- **Tooltip / help text:** Small label displayed above the main title.
- **Input limitations:** Maximum 30 characters.

---

## Title – first part

- **Field label:** Title (first part)
- **Required:** Required
- **Default value:** Mandatory title about the
- **Tooltip / help text:** First part of the main title shown in TT Commons.
- **Input limitations:** Maximum 40 characters.

---

## Title – highlighted part

- **Field label:** Title (highlighted part)
- **Required:** Required
- **Default value:** call-to-action
- **Tooltip / help text:** Highlighted part of the title that uses a different font style.
- **Input limitations:** Maximum 25 characters.

---

## Title – last part

- **Field label:** Title (last part)
- **Required:** Required
- **Default value:** here
- **Tooltip / help text:** Final part of the main title.
- **Input limitations:** Maximum 20 characters.

---

## Body text

- **Field label:** Body text
- **Required:** Optional
- **Default value:** Optional body copy goes here.
- **Tooltip / help text:** Supporting text displayed below the main title.
- **Input limitations:** Maximum 120 characters.

---

## Top button text

- **Field label:** Button text
- **Required:** Optional
- **Default value:** Optional button
- **Tooltip / help text:** Text displayed in the link-style button below the body copy.
- **Input limitations:** Maximum 25 characters.

---

## Top button link

- **Field label:** Button link
- **Required:** Optional
- **Default value:** #
- **Tooltip / help text:** URL where the button should link to.
- **Input limitations:** Must be a valid internal or external URL.

---

# Card 1

## Card 1 title

- **Field label:** Card 1 title
- **Required:** Required
- **Default value:** Title
- **Tooltip / help text:** Title displayed on the first image card.
- **Input limitations:** Maximum 30 characters.

---

## Card 1 description

- **Field label:** Card 1 description
- **Required:** Required
- **Default value:** Supporting text about the call-to-action goes here.
- **Tooltip / help text:** Supporting text displayed on the image card.
- **Input limitations:** Maximum 80 characters.

---

## Card 1 image

- **Field label:** Card 1 image
- **Required:** Required
- **Default value:** None
- **Tooltip / help text:** Image used as the background of the first card.
- **Input limitations:** Landscape image recommended. Minimum width 800px.

---

## Card 1 link

- **Field label:** Card 1 link
- **Required:** Required
- **Default value:** #
- **Tooltip / help text:** Link opened when the first card is clicked.
- **Input limitations:** Must be a valid internal or external URL.
